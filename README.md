# Movie Locker Bot — Bot Only Channel Management

در این نسخه دیگر لازم نیست کانال‌های اجباری را داخل Railway Variables وارد کنی.
ادمین همه کانال‌های اجباری را از داخل خود ربات مدیریت می‌کند.

## Railway Variables لازم

فقط این‌ها لازم است:

```env
BOT_TOKEN=توکن ربات
ADMIN_IDS=آیدی عددی ادمین
DATABASE_URL=sqlite+aiosqlite:////app/data/bot.db
DELETE_AFTER_SECONDS=15
SKIP_MEMBERSHIP_FOR_ADMINS=1
```

دیگر لازم نیست این‌ها را در Railway بزنی:

```env
REQUIRED_CHANNEL
REQUIRED_CHANNEL_LINK
```

## مدیریت کانال‌ها داخل ربات

نمایش پنل کانال‌ها:

```text
/channels
```

افزودن کانال عمومی:

```text
/addchannel @irfreenet
```

جایگزینی همه کانال‌های قبلی با یک کانال:

```text
/setchannel @irfreenet
```

حذف کانال:

```text
/delchannel1
```

برای کانال خصوصی:

```text
/addchannel -1001234567890 https://t.me/+InviteLink
```

## دکمه شیشه‌ای عضویت

برای کانال عمومی، وقتی با `/addchannel @username` اضافه شود، ربات خودش لینک شیشه‌ای را می‌سازد:

```text
https://t.me/username
```

برای کانال خصوصی، چون یوزرنیم عمومی ندارد، باید لینک دعوت را بدهی.

## نکته مهم

ربات باید داخل هر کانال اجباری Admin باشد تا بتواند عضویت کاربران را بررسی کند.

## منوی شناور / برای ادمین

در این نسخه وقتی ادمین داخل ربات `/` بزند، دستورهای مدیریتی را به صورت شناور می‌بیند:

```text
/add
/files
/stats
/channels
/addchannel
/setchannel
/delchannel
/debug_channel
```

نکته: خود تلگرام فقط نام دستور را در منوی شناور نشان می‌دهد، نه آرگومان کامل.  
برای مثال بعد از انتخاب `/addchannel` باید ادامه‌اش را خودت بنویسی:

```text
/addchannel @irfreenet
```

یا برای کانال خصوصی:

```text
/addchannel -1001234567890 https://t.me/+InviteLink
```


## نسخه movie-section-gate-v1

در این نسخه برای کاربران چهار دکمه اصلی اضافه شده است:

- 📢 کانال تلگرام
- 🔐 فیلترشکن پرمیوم رایگان
- 💎 دریافت اکانت های پولی سایت های معروف
- ☎️ ارتباط با تیم ما

هر دکمه کانال‌های عضویت اجباری جداگانه خودش را دارد.

کلیدهای هر بخش:

- `telegram` برای کانال تلگرام
- `vpn` برای فیلترشکن پرمیوم رایگان
- `accounts` برای بخش اکانت‌ها
- `contact` برای ارتباط با تیم ما

نمونه افزودن کانال برای فیلترشکن:

```text
/addsectionchannel vpn @channel1
/addsectionchannel vpn @channel2
/addsectionchannel vpn @channel3
/addsectionchannel vpn @channel4
```

نمونه افزودن کانال برای اکانت‌ها:

```text
/addsectionchannel accounts @channelA
/addsectionchannel accounts @channelB
```

برای دیدن لیست کانال‌های اختصاصی:

```text
/sectionchannels
```

برای حذف کانال اختصاصی:

```text
/delsectionchannel 1
```

نکته: برای بخش اکانت‌ها فقط اطلاعات یا لینک‌هایی را قرار بده که مجوز انتشارش را داری.


## نسخه v2 — دستورهای سریع ادمین برای افزودن کانال

تلگرام اجازه نمی‌دهد متن کامل همراه آرگومان به صورت خودکار داخل باکس تایپ نوشته شود. برای همین در این نسخه دستورهای کوتاه جدا اضافه شده‌اند. ادمین وقتی `/` را بزند این دستورها را می‌بیند:

- `/addvpnchannel`
- `/addaccountchannel`
- `/addtelegramchannel`
- `/addcontactchannel`

روش استفاده سریع:

```text
/addvpnchannel @channel1
/addaccountchannel @channel2
/addtelegramchannel @channel3
/addcontactchannel @channel4
```

اگر ادمین فقط خود دستور را بفرستد، ربات مرحله بعدی را باز می‌کند و فقط اسم کانال را می‌پرسد. برای کانال خصوصی هم این قالب را بفرست:

```text
-1001234567890 https://t.me/+InviteLink
```

بعد از Deploy داخل Railway Logs باید این عبارت دیده شود:

```text
Bot started — movie-section-gate-v2-quick-admin-commands
```


## نسخه v3 — گیت عمومی در /start و ارتباط بدون عضویت جدا

در این نسخه وقتی کاربر `/start` می‌زند، ابتدا کانال‌های اجباری عمومی نمایش داده می‌شود. بعد از تایید عضویت، منوی کاربر نمایش داده می‌شود.

دکمه‌های کاربر:
- 📢 کانال تلگرام
- 🔐 فیلترشکن پرمیوم رایگان
- 💎 دریافت اکانت های پولی سایت های معروف
- ☎️ ارتباط با تیم ما

گزینه ارتباط با تیم ما عضویت اجباری اختصاصی ندارد.

مدیریت کانال عمومی شروع:
- `/addchannel @YourChannel`
- `/channels`
- `/delchannel 1`
- `/debug_channel`

نکته مهم درباره منوی `/` تلگرام:
تلگرام اجازه نمی‌دهد ربات کل دستور را داخل Write Message به صورت سفارشی پر کند. وقتی دستور `/addchannel` خالی ارسال شود، هیچ کانالی اضافه نمی‌شود و فقط نمونه دستور نمایش داده می‌شود. ادمین باید بعد از دستور، نام کانال را اضافه کند؛ مثال: `/addchannel @YourChannel`.

بعد از Deploy، داخل Railway Logs باید این عبارت را ببینی:
`Bot started — movie-start-gate-v3-contact-free`

```text
-1001234567890 https://t.me/+InviteLink
```

## نسخه v5 — پنل شیشه‌ای مدیریت کانال‌های اجباری

در این نسخه ادمین لازم نیست دستور کامل را حفظ کند.

از منوی ادمین دکمه زیر را بزن:

```text
➕ افزودن کانال اجباری
```

بعد ربات می‌پرسد این کانال برای کدام بخش اضافه شود:

- عضویت اجباری شروع ربات
- فیلترشکن پرمیوم رایگان
- دریافت اکانت های پولی سایت های معروف
- کانال تلگرام

بعد از انتخاب بخش، ادمین فقط یوزرنیم کانال را می‌فرستد:

```text
@YourChannel
```

برای مدیریت کانال‌ها هم دکمه زیر را بزن:

```text
📢 کانال‌های اجباری
```

ربات اول بخش‌ها را نشان می‌دهد. بعد از انتخاب هر بخش، لیست کانال‌های همان بخش نمایش داده می‌شود و برای هر کانال گزینه‌های زیر وجود دارد:

- ادیت
- حذف
- افزودن کانال جدید

گزینه ارتباط با تیم ما عضویت اجباری جداگانه ندارد.

بعد از Deploy در Railway Logs باید این عبارت را ببینی:

```text
Bot started — movie-admin-channel-manager-v5
```


## نسخه v6 — رفع لینک فایل و پاکسازی دستورهای شناور ادمین

تغییرات:
- لینک فایل‌های ثبت‌شده دیگر به `REQUIRED_CHANNEL_LINK` وابسته نیست.
- وقتی کاربر روی لینک فایل می‌زند، اگر عضو کانال عمومی نباشد همان کانال‌های اجباری عمومی دیتابیس نمایش داده می‌شود.
- خطای «لینک عضویت کانال تنظیم نشده است» برای لینک فایل حذف شد.
- دستورهای اضافه روش قبلی از منوی شناور `/` ادمین حذف شدند.
- منوی شناور ادمین فقط این موارد را نشان می‌دهد:
  - `/start`
  - `/help`
  - `/add`
  - `/files`
  - `/stats`
  - `/channels`
  - `/addchannel`

نکته:
برای اضافه کردن کانال عمومی شروع ربات از داخل خود ربات استفاده کن:
`/addchannel`
بعد بخش «عضویت اجباری شروع ربات» را انتخاب کن و یوزرنیم کانال را بفرست.


## نسخه v7 — اصلاح دریافت فایل و پاک‌سازی دستورهای قدیمی

تغییرات مهم:
- لینک فایل با `?start=f_ID` از همان کانال‌های عضویت اجباری شروع ربات استفاده می‌کند.
- دیگر برای لینک فایل به `REQUIRED_CHANNEL_LINK` نیاز نیست.
- کاربر چه `/start` خالی بزند، چه از لینک فایل وارد شود، همان عضویت اجباری عمومی را می‌بیند.
- دکمه تکراری «کانال‌های بخش‌ها» از منوی ادمین حذف شد.
- دستورهای قدیمی روش قبلی مثل `/addvpnchannel`، `/addsectionchannel` و `/sectionchannels` از منوی شناور `/` پاک می‌شوند.
- اگر ادمین دستور قدیمی را دستی بفرستد، ربات به پنل جدید مدیریت کانال‌ها راهنمایی می‌کند.

بعد از Deploy، در Railway Logs باید ببینی:
`Bot started — movie-file-gate-commands-clean-v7`

نکته:
اگر تلگرام همچنان دستورهای قدیمی را در منوی `/` نشان داد، یک بار چت ربات را ببند و باز کن یا چند دقیقه صبر کن؛ تلگرام گاهی Command Menu را کش می‌کند.


## نسخه v7.1 — اصلاح دکمه مدیریت کانال‌های اجباری

در این نسخه دکمه «➕ مدیریت و افزودن کانال اجباری» مستقیماً پنل کامل مدیریت کانال‌ها را باز می‌کند.

مسیر کار:
1. دکمه «➕ مدیریت و افزودن کانال اجباری» یا «📢 کانال‌های اجباری» را بزن.
2. بخش مورد نظر را انتخاب کن.
3. داخل همان بخش، لیست کانال‌ها را می‌بینی.
4. کنار هر کانال دکمه‌های «✏️ ادیت» و «🗑 حذف» هست.
5. پایین همان صفحه دکمه «➕ افزودن کانال جدید» هست.

دستور کمکی:
- `/cancel` برای خروج از حالت ثبت فایل یا ثبت کانال
- `/version` برای بررسی نسخه فعال

بعد از Deploy باید در Railway Logs ببینی:
`Bot started — movie-bot-v7.1-channel-manager-button-fix`


## نسخه v7.2 — مدیریت فایل‌ها با دکمه

در این نسخه بخش «📋 لیست فایل‌ها» برای هر فایل دکمه دارد:

- ✏️ ادیت عنوان
- ⛔️ غیرفعال کردن / ✅ فعال کردن
- 🗑 حذف کامل

نکته:
- غیرفعال کردن یعنی لینک فایل باقی می‌ماند، اما کاربر فایل را دریافت نمی‌کند.
- حذف کامل یعنی رکورد فایل از دیتابیس حذف می‌شود و لینک آن دیگر کار نمی‌کند.

بعد از Deploy باید در Railway Logs ببینی:
`Bot started — movie-bot-v7.2-file-manager-buttons`

برای تست نسخه:
`/version`
